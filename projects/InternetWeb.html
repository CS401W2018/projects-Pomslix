Internet

Definition: The Internet is a series of communication network used by computer and/or servers to connect and share information from one to another around the world.

Web

Definition: The Web are web pages or documents that usually anyone can access. Connecting to a web page usually requires internet to function and display the page for the user.
To get to a specific web page, users need the URL for the website or a Hyperlink.

Internet and Web Differnt

Explaination:Internet allows computers to communicate with other networks to share information while the Web pages are places where these information can be shared for easier access

ASCII

Definition: ASCII is the American standardized encoding format for computers. ASCII uses numbers to assign value to numbers or alpbetical letters

Unicode

Definition: Unicode is the International standardized encoding format for computer. It is similar to that of ASCII but has support for different languages.

UTF-8

Definition: UTF-8 is a standard encoding for Unicode that allows the unicode to be translated into binary codes. It is coded between one to four 8-bit bytes.

URL

Definition: The URL is a web address that is located at the top of web page,just below the web tab.

Domain Name

Definition: The domain name is a part of the URL that identifies what the website and usually has a unique address that are easy to remember.

HTTP

Definition: HTTP stands for Hypertext Transfer Protocol and is used to allow information to be shared across the internet and websites.

HTTPS

Definition: Like HTTP, It allows information to be shared but also make the web browser and websites sharing of information secured.

IP Address

Definition: A IP Address is a strand of numerical value that allows Computers to access the internet and communicate across the global network.

DNS

Definition: DNS stands for Domain Name System and is used to translate domain names into IP Address so the user's web browser can access it and load information stored on the website.

W3C Recommendation

Definition: W3C known as World Wide Web COnsortium is an international group that developes the rules and standards of the World Wide Web so that developers and people alike can have consistent experiences

XML

Definition: XML is a markup language that is able to describe,store, and transport data

XHTML

Definition: XHTML is an extension of HTML that allows it to work with XML for more flexibility.

HTML

Definition:HTML is the standardized markup language for creating basically any web pages on the world wide web.

HTML5

Definition: HTML5 is basically the same as HTML, but is more modernized so that developers can read and understand it more easily.
