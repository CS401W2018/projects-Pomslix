Inline css is when you use style="#" in a tag. Embedded tag is when you use <style> in the head tag.
External css is when you use a <link> in the head tag and link it to a external css file.

You should only use a external css because it is more easier to manage. You also use it because when working on a webpage with a team,
programmers usually code the html and graphic designers manage the css for web designs.

Selectors are used to target a specific tag you want to affect and change it according to your web page style by changing the color, font, weight, etc.
